Usage

Keep it consistent.

Do

  • Use the official PNG downloaded from this page
  • Give it breathing room and don't crowd it against other elements
  • Write SOLARIS in all caps when referencing the brand name
  • Use it on both dark and light backgrounds, the transparent PNG works for both
  • Scale it proportionally

Don't

  • Recolour it or swap the gradient for something else
  • Stretch, squash, or skew it
  • Add drop shadows, glows, or outlines to the logo itself
  • Drop it on a busy background where it gets lost
  • Recreate it from scratch or modify the shape
