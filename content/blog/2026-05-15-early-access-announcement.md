# Early Access Launches June 27th

Everything we have been building has led up to this moment.

**SOLARIS Early Access launches on June 27th, 2026 at 4:00 PM EST.**

## Season 0: First Light

Early Access will run as Season 0, titled *First Light*, through the end of July. This is the beginning of Solaris — a chance to play, shape, and experience the server before it opens to the world.

To mark the occasion, we will be hosting a live stage starting roughly 30 minutes before the server goes live. The stage will run for approximately one hour. If you want to be there for the launch moment, stay tuned for more details.

## Getting In

There are a few ways to secure Early Access:

**Purchase:** Anyone who supports the server with £4.99 or more will receive Early Access immediately when the server opens.

**Discord Boost:** Boosting our Discord server will also grant you access.

**Waitlist:** Sign up to the waitlist at [playsolaris.net](https://www.playsolaris.net). We will be periodically whitelisting players from the email list at random, so it is worth signing up even if you are not planning to pay upfront.

## About Progress During Early Access

Progress made during Early Access is **not permanent**. When the server moves to its public release, progress resets to ensure a fair start for everyone.

The one exception is supporter items. Exclusive cosmetics and rewards earned during Early Access are yours to keep permanently as a thank-you for being there early.

## Season 1: Starfall

Season 1 of SOLARIS, titled **Starfall**, begins on **August 1st, 2026** and is completely free for all players.

At the start of Season 1, we will also be running the **BETA Unobtainable Event** — a limited-time event featuring exclusive items that will never return. If you are an Early Access supporter, this is part of what you are playing for.
